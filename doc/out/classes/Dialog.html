Dialog Class

Defined in: event-calendar.js:339
Module: Calendar

Implements modal dialog which contains selected start and end date, component with events to be selected and inputs where user can type some message for other person and/or note for himself. There is available both Submit and Delete button and the cross for closing dialog without applying any changes.
